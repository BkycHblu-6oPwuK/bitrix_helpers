# как развернуть проект локально

1. скачать конфиг под докер из ./docky, используйте docky для удобства - https://github.com/BkycHblu-6oPwuK/docky
2. .env.example переименовать в .env
3. развернуть резервную копию через restore.php
4. docky down
5. в .env докера установить переменной NODE_PATH путь /var/www/nuxt, там весь фронтенд, в целом это можно сделать сразу если создадите папку nuxt самостоятельно, чтобы докер при запуске сам этого не сделал с правами root
6. docky up -d
7. установите зависимости - docky npm i, docky composer i

далее в /nuxt/ проверить файл .env, если не создан, то создайте его на основе .example

далее docky npm run dev должен запустить сервер разработки и можно на нем работать
